# 🎫 Helpdesk & Ticket Management System

מערכת ניהול פניות (Helpdesk) מקצועית ומקיפה המבוססת על **React** ו-**TypeScript**. המערכת מאפשרת ניהול מלא של מחזור חיי פנייה (Ticket Lifecycle), תוך הפרדה קפדנית של הרשאות וגישה בהתאם לתפקיד המשתמש.

---

## 🚀 סקירה כללית (Project Overview)

הפרויקט נבנה כפתרון לניהול מערך תמיכה טכנית, המקשר בין לקוחות הקצה לבין צוות התמיכה והנהלה. המערכת כוללת ממשק משתמש דינמי המשתנה בזמן אמת בהתאם לזהות המשתמש המחובר.

## ✨ תכונות מרכזיות (Key Features)

### 👥 ניהול הרשאות ותפקידים (RBAC)
המערכת מנהלת שלושה סוגי משתמשים בעלי הרשאות שונות:
* **Customer (לקוח):** פתיחת פניות חדשות כולל הגדרת נושא, תיאור ורמת דחיפות.
* **Agent (סוכן תמיכה):** צפייה בפניות המשויכות אליו, עדכון סטטוס הפנייה (פתוח, בטיפול, סגור) והוספת תגובות.
* **Admin (מנהל):** שליטה מלאה על כל הפניות במערכת, הקצאת סוכנים לטיפול בפניות וניהול רשימת המשתמשים.

### 🛠️ ניהול פניות חכם
* **דשבורד דינמי:** כותרות ופעולות מותאמות אישית לכל תפקיד להגברת היעילות בעבודה.
* **סינון וחיפוש:** אפשרות לחיפוש חופשי לפי נושא וסינון מתקדם לפי סטטוס (איחוד לוגי של מצבי "בטיפול" ו"בבדיקה").
* **צ'אט תמיכה:** מערכת תגובות המאפשרת תקשורת רציפה בין הלקוח לסוכן בתוך דף הפנייה.

### 🔐 אבטחה ותקשורת
* **Authentication:** תהליך התחברות מאובטח מבוסס **JWT Token**.
* **Context API:** ניהול מצב גלובלי השומר על סנכרון המשתמש בכל חלקי האפליקציה.
* **API Wrapper:** שירות מרכזי לניהול בקשות HTTP המטפל אוטומטית בכותרות האבטחה (Bearer Token) ובשגיאות שרת.

## 🛠️ טכנולוגיות (Tech Stack)

* **Frontend:** React 18, TypeScript, Vite.
* **UI Library:** Material UI (MUI).
* **Forms:** React Hook Form.
* **Notifications:** SweetAlert2 & MUI Snackbar.
* **State Management:** Context API & useReducer.
